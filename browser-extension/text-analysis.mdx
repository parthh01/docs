## Using Text Analysis

<Steps>
  <Step title="Select Text">
    Highlight any text on a webpage
  </Step>
  
  <Step title="Analyze">
    Right-click and select "Analyze with Critique AI"
  </Step>
  
  <Step title="Review">
    View detailed analysis in the sidebar
  </Step>
</Steps>

## Analysis Features

### Content Understanding
- Main topics and themes
- Key arguments and claims
- Sentiment analysis
- Bias detection

### Citation Verification
- Source credibility
- Supporting evidence
- Related research
- Expert opinions

### Context Analysis
- Historical background
- Current relevance
- Related discussions
- Alternative viewpoints
