## Overview

Chat with Tabs allows you to have intelligent conversations with Critique AI using your open browser tabs as context, enabling:
- Cross-reference information across multiple sources
- Get comprehensive answers based on current research
- Verify claims against multiple sources
- Generate insights from combined contexts

### Tab Selection
- Choose specific tabs for context
- Automatically include relevant tabs
- Filter by domain or content type
- Prioritize tabs by relevance

### Conversation Capabilities
- Ask questions about content across tabs
- Compare information between sources
- Generate summaries of multiple articles
- Find contradictions or confirmations

### Context Management
- Save important conversations
- Export citations and sources
- Share insights with collaborators
- Bookmark relevant sections
