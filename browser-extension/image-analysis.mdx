## Using Image Analysis

<Steps>
  <Step title="Access">
    Right-click any image on the web
  </Step>
  
  <Step title="Analyze">
    Select "Critique This" from the context menu
  </Step>
  
  <Step title="Review">
    View detailed analysis by hovering over the tooltip.
  </Step>
</Steps>

## Analysis Features

### Content Verification
- Source identification
- Manipulation detection
- Date verification
- Context analysis

### Visual Understanding
- Object recognition
- Scene description
- Text extraction
- Brand identification

### Source Tracking
- Original publication
- Image history
- Usage context
- Similar images
