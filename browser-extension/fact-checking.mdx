## How It Works

Critique AI automatically analyzes content as you browse, providing:

- Credibility scores
- Source verification
- Claim analysis
- Citation links

## Features

### Automatic Detection
Content is analyzed in real-time as you scroll, with visual indicators showing:
- Verified claims (green)
- Disputed claims (yellow)
- Misleading content (red)

### Detailed Analysis
Hover over any indicator to see:
- Supporting evidence
- Contradicting evidence
- Related context
- Original sources

### Custom Verification
Highlight any text, right-click, and select "Critique This" to manually trigger fact-checking for:
- Specific claims
- Quotes
- Statistics
- Historical information
