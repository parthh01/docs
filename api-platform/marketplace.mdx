## Overview

The Critique AI Marketplace is your hub for discovering and integrating community-created APIs. This guide will help you navigate and make the most of the marketplace features.

## Browsing the Marketplace

### Search and Filters

The marketplace offers powerful search and filtering capabilities to help you find the right APIs:

<CardGroup cols={2}>
  <Card title="Search Bar" icon="magnifying-glass">
    Search APIs by title or description to quickly find what you need
  </Card>
  
  <Card title="Advanced Filters" icon="sliders">
    Filter APIs by:
    - Number of active users
    - API call volume
    - Response latency
    - Categories
  </Card>
</CardGroup>

### Featured APIs

At the top of the marketplace, you'll find our featured APIs section showcasing:
- Verified and trusted APIs
- High-performing integrations
- Community favorites

### Trending APIs

The trending section highlights popular APIs based on:
- Active user count
- Usage statistics
- Community ratings

## Understanding API Cards

Each API in the marketplace displays key information:

<Properties>
  <Property name="Title & Description">
    Basic information about the API's purpose and functionality
  </Property>
  
  <Property name="Category">
    The API's primary classification
  </Property>
  
  <Property name="User Count">
    Number of active users integrating the API
  </Property>
  
  <Property name="Rating">
    Community rating based on user reviews
  </Property>
</Properties>

## Navigation Tips

<Steps>
  <Step title="Browse Categories">
    Use category filters to narrow down APIs by their primary function
  </Step>
  
  <Step title="Compare Metrics">
    Review user counts, response times, and ratings to evaluate APIs
  </Step>
  
  <Step title="View Details">
    Click "View this API" to see the API instructions, input and output schema, and to test the API with sample inputs.
  </Step>
  
  <Step title="Access Dashboard">
    Use the Dashboard button to manage your API subscriptions and keys
  </Step>
</Steps>

## Pagination

The marketplace displays 9 APIs per page. Use the pagination controls at the bottom to:
- Navigate between pages
- View total number of available APIs
- Browse through the complete catalog

<Note>
  Your API browsing preferences and filters will persist as you navigate through pages.
</Note>
